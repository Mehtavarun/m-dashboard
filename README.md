```html

    <div id="status-new" class="status-dot"></div>
```

### Updating Health Checks
Add your new service URL to the `runChecks()` function in the script section of `index.html`:

```javascript
checkHealth('https://new-app.mehta.local', 'status-new');
```

## üê≥ Docker Deployment

This repo is built automatically by the `gateway` compose file.
To build it manually:

```bash
docker build -t dashboard .
docker run --name dashboard -p 8080:80 dashboard
```

In case the status light never turns green you probably need to trust the certificate generated by caddy. I used below to command to copy the certificate from docker container and trust it in the system.

```
 docker cp gateway:/data/caddy/pki/authorities/local/root.crt ./homelab-root.crt
```

```
Locate the file homelab-root.crt in Finder (it should be in your current folder).

Double-click it. This will open Keychain Access.

It might ask which keychain to add it to. Choose System (or "login" if System isn't an option).

Find the certificate named "Caddy Local Authority" in the list.

Double-click it to open settings.

Expand the Trust section.

Change "When using this certificate" to Always Trust.

Close the window (you will need to enter your Mac password/TouchID).
```
